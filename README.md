# AC2WD

This is the Authority Control to Wikidata (AC2WD) project. It is an API that can "translate" various Authority Control (AC) records to Wikidata items. You can find the live API [here](https://ac2wd.toolforge.org). An accompanying user script to use on Wikidata is [here](https://www.wikidata.org/wiki/User:Magnus_Manske/ac2wd.js).
